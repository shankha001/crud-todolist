# CRUD - Todolist

Simple Todolist using NodeJS and MongoDB displaying CRUD(Create, Read, Update, Delete) functionality.


## Dependencies

* body-parser
* dotenv
* ejs
* express
* mongoose

## Installation

For setting up the project locally

```sh
$ npm install
$ npm start
```
